<h1>🚀 Scalable Task Management API System</h1>

A robust backend application built during my internship at Webito, designed to manage tasks efficiently using RESTful APIs. This project demonstrates real-world backend development practices including modular architecture, API design, and database integration.

<h2>📌 Overview</h2>

This project was developed after learning core backend concepts and applying them practically in March. It focuses on building a scalable and maintainable system for managing tasks and comments.

<h2>✨ Features</h2>
Create, update, delete, and retrieve tasks (CRUD operations)
Add and manage comments on tasks
Modular and scalable backend architecture
RESTful API design
Error handling and validation

<h2>🛠️ Tech Stack</h2>

1.Backend: Node.js, Express.js

2.Database: MongoDB 

3.Tools: Postman, Git, GitHub

<h2>⚙️ Installation & Setup</h2>
1. Clone the repository
git clone https://github.com/Jainishd24/Scalable-Task-Management-API-System.git

2. Navigate to project folder
cd Scalable-Task-Management-API-System

3. Install dependencies
npm install

4. Create .env file
Add your environment variables:
PORT=5000
DATABASE_URL=your_database_url

5. Run the server
npm start
